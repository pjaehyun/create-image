# 📌 이미지 변환 자동화 프로그램

이미지 URL을 이미지 파일로 변환해주는 프로그램
</br>
</br>

# 1. 제작 기간

2023.02.23 ~ 2023.02.24
</br>
</br>

# 2. 사용법

1. "대표이미지" 칼럼이 존재하는 액셀파일을 선택한다.
2. 변환된 이미지를 저장할 폴더를 선택한다.
3. 원하는 픽셀로 설정한다.
4. 변환버튼을 누른다.
   </br>
   </br>

# 3. 사용기술

- Python3
- PyQt6
- Pandas
- Pillow
- Pyinstaller
  </br>
  </br>
